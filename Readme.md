# Backend del Sistema de Recomendación Turística

Este documento explica cómo configurar y ejecutar el backend del sistema de recomendación de lugares y eventos turísticos.  
El backend está desarrollado en **Python (FastAPI)** y utiliza **MongoDB** como base de datos.

---

## 1. Requisitos previos

Asegúrate de tener instalados los siguientes componentes:

- [Anaconda / Miniconda](https://docs.conda.io/en/latest/miniconda.html)
- [Git](https://git-scm.com/)
- [Docker](https://www.docker.com/) *(Para despliegue)*
- Python ≥ 3.11 (si no usas Conda)
